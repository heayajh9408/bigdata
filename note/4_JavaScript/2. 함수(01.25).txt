*함수
1. 익명함수
 - 함수의 이름이 없는 함수로 변수에 넣어 사용
 - 익명함수 호출 시 함수형 변수에 할당 후 변수 이름에 ()을 붙여 호출
2. 선언적 함수
 - 이름이 있는 함수를 말하며 function 함수명 () {} 의 형태
 - 선언적 함수는 정의 전에 사용이 가능하지만 익명함수는 불가
3. 매개변수와 리턴값
 - 선언된 매개변수보다 많은 매개변수 뒷부분은 무시
 - 선언된 매개변수보다 적은 매개변수는 undefined
 - Array()함수; 빈 배열 생성 // Array(number); 매개변수만큼의 크기를 가지는 배열 생성
 - Array(any, ..., any); 매개변수를 배열로 생성
4. 가변인자함수
 - 매개변수의 갯수에 따라 변할 수 있는 함수
 - 자바스크립트에서 배열의 방의 갯수는 가변적
 - push를 통해 배열방 추가가 가능하고 pop을 통해 삭제가 가능
 # arguments에서는 forEach() 사용 불가
5. 내부함수
 - 함수 내부에 선언하는 함수
 - 충돌을 막기 위해 내부함수를 사용하면 내부함수가 우선
6. 자기호출함수
 - 다른 개발자에게 영향을 주지 않게 함수를 생성하자 마자 호출하는 경우
 - ex) (function () {alert('자기호출함수'); })();
7. 콜백함수
 - 함수도 하나의 자료형으로 매개변수에 넣을 수 있는데 매개변수에 넣어 전달된 함수를 말함
8. 클로저함수
 -  특정함수를 여러 번 수행 또는 처리하는 경우 필요한 리턴하는 함수
9. 타이머함수(반복문에 유의)
 - setTimeout(function, millisecond); 일정시간 후 함수를 한 번 실행
 - setInterval(function, millisecond) ; 일정시간마다 함수를 반복해서 실행
 - clearTimeout(id) ; setTimeout 중지
 - clearInterval(id); setInterval 중지
10. 인코딩과 디코딩함수
 - 문자를 부호화하여 전송(인코딩), 그 반대로 돌리고 해석(디코딩)
