*객체
 - 여러가지 변수를 모아놓은 자료형, typeof() 이용 시 object
 - 배열 내부에 있는 값을 요소(element), 객체 내부에 있는 값은 속성(property)라고 하며
   자바 스크립트에서는 요소와 속성을 구분하지 않고 사용
 - 객체 속성 중 함수형 자료형인 속성을 특별히 메서드라고 부름

*객체와 반복문
 - 배열은 단순 for 반복문, for-in 반복문으로 요소에 쉽게 접근
 - 객체는 단순 for문은 불가하고 for in 반복문을 사용

 #with 키워드
 -  output += 'name : ' + student.name + '\n'; output += 'kor : ' + student.kor + '\n';
     --> with(student){output += 'name : ' + name + '\n'; output += 'kor : ' + kor + '\n';}
 - with 키워드와 객체가 충돌하는 경우 window 사용

*생성자 함수
 - 객체지향 프로그램에서 객체를 생성할 때 호출되는 단위 메서드
 - new 키워드로 객체를 생성하고 이러한 생성자 함수로 만든 객체를 객체 또는 인스턴스
   라고 부르며 instanceof 키워드를 사용하여 생성자 함수를 확인

*프로토타입
 - 생성자 함수로 생성된 객체가 공통으로 가지는 공간
 - 프로토타입은 만드는 것이 아니라 arguments처럼 자바스크립트의 모든 함수는
   prototype 객체를 가짐

*캡슐화
 - 특정 속성이나 메서드를 사용자가 바로 사용할 수 없게 막는 기술
 - 이 때 숨겨놓은 특정 속성은 getter와 setter를 이용하여 간접적으로 접근

*상속
 - 기존의 생성자 함수는 객체를 기반으로 새로운 생성자 함수나 객체를 쉽게 만드는 것
