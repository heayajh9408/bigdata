*R 데이터 종류 및 구조

1. 데이터 속성에 따른 구분
 o 명명식(Nominal): 명목척도를 나타내며 이름으로 명명되는 자료 ex)성별(남,녀)
 o 순서식(Ordinal): 서열척도를 나타내며 순서가 있는 명명식 ex)상중하
 o 구간식(Interval): 간격척도를 나타내며 순서의 간격을 측정할 수 있는 순서식 ex)온도
 o 비율식(Ratio): 비율척도를 나타내며 절대 영점이 존재해 비율이 의미 있는 구간식 ex)체중

2. 변수 타입에 따른 구분
 o 연속형: 체중처럼 연속적인 값을 갖는 변수
 o 이산형: 동전의 앞, 뒤처럼 값의 개수가 정의된 변
 o 범주형: 연령대 10대, 20대 처럼 연속형 변수를 구간으로 묶어서 만든 변수

3. 데이터 품질에 따른 구분
 o 특이값(Outlier): 정상적이지 않은 데이터로 잘못 측정된 값이나 오차에 의한 값
 o 결측값(Missing): 측정되지 않은 값으로 R에서는 NA로 표기

4. 분석용변수
 o 요약변수: 데이터 분석을 위해서 1차 가공한 변수
  - 연속형 변수를 구간화하여 가공하거나 시계열 데이터를 기간별, 요일별 등으로 가공한 데이터
 o 파생변수: 분석자의 판단에 따라 특정 조건을 만족하는 변수
  - 가중치를 부여한 변수들 ex)근무시간 구매지수, 주 구매 매장 등

*Factor
 - 팩터(Factor, 요인)는 범주형(Categorical)변수를 의미
 - 미리 정해진 여러 개의 값 중 하나의 값을 가짐
 - levels() :범주의 목록을 출력 ex) gender <- factor(c("남", "여",), levels= c("남","여"))

*구조형 변수와 복합형 변수
 o 구조형 변수: 동일한 데이터타입의 데이터만 저장할 수 있는 변수(배열, 행렬)
 o 복합형 변수: 서로 다른 데이터타입의 데이터를 저장할 수 있는 변수(리스트, 데이터테이블)

* Vector(벡터)
 o 여러 개의 동일한 형태의 데이터를 모아서 함께 저장되는 세트 또는 집합
  - 함수 안에 나열되는 인자들은 한 가지 유형의 스칼라 타입이여야 한다
  - 문자, 숫자, 논리 타입이 있을 경우 문자 타입,  숫자와 논리타입이 있을 경우 숫자로 자동  형변환
  - character()는 문자열 , numeric()은  숫자, logical()은 논리 값을 저장하는 벡터 생성
  - c() 함수를 통해 여러 벡터를 하나의 벡터로 결합할 수 있음
  - append() 새로운 벡터 추가, union() 합집합, intersect() 교집합, setdiff() 차집합, setequal() 비교

* List(리스트)
 o 복합 구조형의 벡터에 해당하는 데이터 타입
  - list(키=값, 키=값,..) 형태로 데이터를 나열해 정의

* Matrix(행렬)
 o 행과 열을 가지는 2차원 배열로 벡터와 마찬가지로 한 가지 유형의 스칼라 데이터들만 저장 가능
  - 열 우선으로 채워지며 행 우선을 원할 경우 byrow=TRUE 속성 사용
  - 행과 열에 명칭 부여시 dimnames() 사용
  - 행렬의 차원은 ncol() 또는 nrow() 사용
  - 행렬의 곱은 %*%를 사용하며 전치행렬은 t(), 역행렬은 solve(행렬)로 계산

* Array(배열)
 o 3차원 이상의 데이터를 다룰 경우에 사용하며 dim 속성을 이용해 차원의 크기를 지정
  - ex) dataArray <- array(data, dim=c(행의수, 열의수, 면의수))
  - attr()함수를 이용하여 객체의 속성을 변경 ex) attr(객체, "속성이름") 

* 데이터 프레임
 o 2차원 구조이며 복합형 데이터타입으로 처리할 데이터를 마치 엑셀의 스프레드 시트와같이 표형태로
    정리한 모습이다
  - str()함수로 구조파악이 가능하며 names()함수로 열 이름을 새로 지정할 수 있음
  - cbind() 열 단위 병함, rbind 행 단위 병합
  - [ ]와 색인을 이용하여 원하는 위치의 열과 행 정보를 조회, 색인을 음수로 표현하면 해당 행 또는 열 제외
  - subset을 이용하여 원하는 정보 조회
