*R_프로그래밍

 o if ~else : 다른 언어와 동일
 o ifelse()함수
  - 간단한 if~else 블록을 작성할 때 사용
  - ifelse(test, true 시 리턴 값, false 시 리턴 값
 o switch(statement, list): state 명령문이 평가되고 이 값에 따라 list의 해당 항목 반환
 o 반복문 - for: 벡터의 데이터를 모두 소비할 때까지 실행하는 명령문
 o 반복문 - while: 다른 언어와 동일
 o 반복문 - repeat: 루프를 종료시키는 조건 문장이 없기 때문에 내부에 종료조건 명시 및 break 필요
 o 탈출문 - next: 반복문에서 아무것도 하지 않고 다음 반복으로 진행
 
 o중위 연산자
  - %%: 나눗셈 후 나머지 , %/%: 몫, %*%: 행렬의 곱 또는 벡터의 내적
  - %o%: 외적, %in%: 매칭연산자

 o리턴문
  - return(expression): 함수에서 리턴된 값은 유효한 오브젝트가 될 수 있음
  - return()문이 없는 함수에서는 명시적으로 반환된 값이 없을 경우 마지막으로 평가된 표현식의 값이 자동으로 반환된다
  - return() 함수는 단일 객체만 반환할 수 있으며 여러 값을 반환 시 list객체를 사용

 o 가변인자
  - ex) mean(x, ...) : ...에 의한 추가 인수는 다른 함수로 전달되거나 전달인자로 다양한 값을 전달할 수 있다

 o 재귀 호출
  - 자기 자신을 호출하는 함수이며 문제를 더 작고 간단한 하위 문제로 분해하여 해결할 수 있음

 o 유효범위
  - 변수들은 어디에 선언되어 있는지에 따라 다른 유효 범위를 갖는다
  - 전역변수: 프로그램을 실행하는 동안 존재하는 변수로 어느 부분에서든지 변경 또는 액세 스할 수 있음
  - 지역변수: 함수처럼 프로그램의 특정 부분에만 존재하며 함수 외부에서는 액세스 할 수 없음
