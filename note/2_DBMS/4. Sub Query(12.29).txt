*Sub Query
 o 하나의 SQL 문장의 절 안에 포함된 또 하나의 SELECT 문장
 o 반드시 괄호로 둘러 쌓아야 함
 o 단일행 서브쿼리와 다중행 서브쿼리가 있다

*단일행 서브쿼리
 o 수행 결과가 오직 하나의 행만을 반환하는 서브쿼리이다
 o 연산자들
  - '=', '>', '>=', '<', '<=', '<>'
 o 다중열 서브쿼리도 가능

*다중행 서브쿼리
 o 서브쿼리에서 반환되는 결과가 하나 이상의 행일 때 사용하는 서브쿼리
 o 연산자들
  - IN: 메인 쿼리의 비교 조건이 서브 쿼리의 결과 중 하나라도 일치하면 참
  - ANY, SOME: 메인 쿼리의 비교 조건이 서브 쿼리 결과 중 하나 이상이 일치하면 참
  - ALL: 메인 쿼리의 비교 조건이 서브쿼리의 모든 값과 일치하면 참
  - EXISTS: 메인 쿼리의 비교 조건이 서브 쿼리의 결과 중 하나라도 존재하면 참
 o 다중 열 서브쿼리도 가능
