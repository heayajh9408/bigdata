*데이터베이스란?
 - 여러 응용 시스템들이 공유할 수 있도록 통합, 저장된 운영데이터의 저장소
 - 운영상 필요한 데이터를 중복을 최소화하여 컴퓨터 기억 장치 내에 모아놓은 데이터 집합

*데이터베이스관리시스템(DBMS)란?
 - 데이터베이스를 공유할 수 있도록 관리, 유지해주는 일련의 소프트웨어시스템

*DBMS의 기능
 - 정의기능: 데이터의 형태, 구조, 저장에 관한 내용을 정의
 - 조작기능: 요구에 따라 검색, 갱신, 삽입, 삭제 등을 지원하는 기능
 - 제어기능: 데이터베이스 내용에 대해 정확성과 안정성을 유지하는 기능

#DBMS의 단점
 - 운영비(초기 운영비, 유지보수비 등등)가 많이 든다
 - Backup / recovery 가 어렵다(구조가 복잡하여 장애 발생시 정확한 이유, 상태 파악 어려움)
 - 시스템의 취약성(일부의 고장이 시스템 전체를 마비시켜 신뢰성과 가용정 저해 가능)

# 데이터 사전(Date Dictionary / System Catalog)이란?
 - 데이터베이스를 운영하기 위해 필요한 정보들의 집합

 - 가장 많이 쓰이는 예로 Select 문이 있다.
 - ex) SELECT [DISTINCT] {*, colum[Alias],--} FROM 테이블명 WHERE 절 ORDER BY~
