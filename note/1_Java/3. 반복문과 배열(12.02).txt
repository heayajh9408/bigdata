*반복문
- 반복적으로 수행하도록 하는 동작 (!!무한 loop에 빠지는 것을 조심!!)

1. for문
- 설정된 횟수만큼 반복적으로 수행
- ex) for(int i=1; int<5; int++){
              System.out.println(i+". 안녕하세요");}

2.while문
- 조건이 만족될 때까지 반복적으로 수행
- ex) while(i>=5){
                조건이 참일 때 계속 실행 명령문}

3.do-while문
- 최초 한번 무조건 실행 후 조건이 만족되면 수행 아니면 수행하지 않음
- ex) do {
              최초 한 번은 무조건 실행, 두 번째부터 참이면 실행할 명령문
           }while(조건식)
