*상속과 인터페이스

상속이란? {( 클래스이름(자식) extends 상위클래스(아빠)}
- 상속이란 쉽게 말해 기존의 클래스를 재사용해서 새로운 클래스를 작성하는 것
- 두 클래스를 부모와 자식으로 관계를 맺어줄 수 있음
- 단, private 멤버는 직접 제어할 수 없다(getter,setter 사용)
- 상위클래스(S), 하위클래스 (A) 경우 S b = new A는 가능, 반대의 경우 불가능

*오버라이드와 오버로딩

1. 오버라이드란?
 - 부모클래스의 메소드를 자식 클래스에서 재정의 하는 것

2. 오버로딩이란?
 - 매개변수의 수나 타입으로 같은 이름의 함수를 중복 정의하는 것
