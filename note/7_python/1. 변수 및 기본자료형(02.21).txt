*변수
 o 프로그램이 실행되는 동안 상황에 따라 변하는 값을 저장
  - 변수의 이름은 문자, 숫자, 밑줄(_)문자를 포함
  - 변수는 숫자로 시작할 수 없고 대소문자를 구분
  - 공백, 문장부호, 특수문자 등은 사용 불가하며 파이썬의 예약어는 사용할 수 없음
  - a, b =10, 20 처럼 한 라인에 여러 개 변수를 선언할 수 있음
  - input("") 함수를 통해 사용자로부터 값을 입력받을 수 있음
  - print('message', sep=' ', end='\n', file=sys.stdout) 를 통해 화면 출력

* 기본 자료형
 o 숫자형
  - 정수(int), 실수(float), 복소수(complex) 형
  - 정수형과 정수형의 나눗셈 연산은 실수형, ex) 2/2 = 1.0
 o 문자형
  - 단일문자와 문자열 구분 없음
  - name = "JinKyung"  address= '서울시 강남구'   print(name, address) / JinKyung 서울시 강남구
  - '''~~~~~~''' -> 줄바꿈도 그대로 적용해서 여러 줄의 문자를 작성
  - [start:stop:step]을 이용하여 문자열 슬라이싱이 가능
 o 논리형
  - True or False 값을 말하며 true 또는 TRUE 로 사용이 불가
  - False로 간주되는 것 : 0, None, ""[], (), {}
  - True로 간주되는 것 :  ' ', 널문자('\0')은 True로 간주
