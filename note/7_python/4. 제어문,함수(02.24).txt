* 제어문
 o if, for - in , while, 중첩 가능, 중첩 for문 안에서의 break
 o 조건문
  - if ~ : ~ elif~ : ~ else: ~ 문법으로 사용, 그 외 다른 언어와 유사
 o 반복문
  - for 변수 in 나열 가능한 자료	
  - while 조건(무한 반복문에 빠지지 않도록 유의)	
 o break, continue
  - break; -> 현재 반복문을 빠져나감
  - continue; -> 반복문의 조건으로 이동
 o 2차원 이상의 데이터 구조의 모든 반복를 위해 중첩루프(반복문 안에 반복문) 방법 사용
 
* 함수
 o 반복해서 사용할 코드를 함수로 정의해  훨씬 구조적이고 간결한 코드를 작성할 수 있다.
  - def 함수명([param1, param2, ...])
    	expressions
  - 기본값을 갖는 매개변수는 뒤에 표시해야 함
 o docstring
  - 함수의 설명서 같은 역할로 주석보다 더 많은 기능을 함
  - ex) def my_function(): """ ~~~~~~~"""
 o 지역변수와 전역변수
  - 지역변수: 함수 안에 정의된 변수들
  - 전역변수: 함수 밖에 정의된 변수들
  - 렉시컬 특성: 함수 안에 변수 선언 시 지역변수를 선언하면서 값을 할당
  - 변수의 값을 조회하는 순서: 로컬 심볼 테이블 -> 전역 심볼 테이블 -> 내장된 이름 테이블
  - return을 이용하여 함수의 실행 결과를 반환할 수 있음
