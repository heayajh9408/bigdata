* 함수의 매개변수
 o 매개변수가 기본값을 가질 때 : 기본값을 갖는 매개변수는 선택적으로 사용 가능
 o 매개변수의 기본 값이 변수일 때 : 함수가 정의되는 시점의 값이 기본값으로 사용
 o 매개변수의 기본값이 변경 가능한 객체일 경우 : 계속
 o 키워드 인수 : 함수 호출시 인수가 매개변수 이름을 갖는 것. 키워드인수는 순서 인수 뒤에
 o 튜플 매개변수( *args)
 o 딕셔너리 매개변수(**kwargs)
 o 매개변수와 인수의 순서 : 순서인수, 튜플인수, 키워드 인수, 딕셔너리 인수
 o 튜플 인수 언패킹  : *변수
 o 딕셔너리 인수 언패킹 : **변수
 o int, float, range 등 import 없이 사용 가능한 내장함수들도 있음

* 람다식
 o 람다식은 작은 익명함수를 의미하며 실행할 문장을 한 문장만 작성
  - return 구문이 없어도 statement의 결과를 반환
  - ex) (lambda a,b : a+b)(30,40)

 o 리스트컴프리헨션, filter(), map()함수
  - map() 함수: list(map(lambda x : 2*x, list_data))
  - 리스트 컴프리헨션: [x*2 for x in list_data]
  - filter() 함수: list(filter(lambda x: x%2==0, list_data))
  - return 문에 람다식을 사용할 수 있음
