01/22(자바와의 차이점 위주)

*기본문법
 - 출력은 alert()로 한다
 - Undefined : 변수를 선언하지 않았거나 선언했어도 초기화하지 않았을 때
 - confirm: 사용자의 확인을 받기 위한 대화상자를 표시
 - prompt: 사용자로 내용을 입력받을 수 있는 대화상자를 표시
 - Number(): 다른 자료형을 숫자로 바꿀 때, 일반 문자열을 숫자로 표시할 수 없지만
                  강제로 숫자자료형으로 바꿨을 시 NaN으로 출력
 - eval: 문자열을 수식으로 바꿀 수 있음
 - isNaN: 주어진 데이터가 숫자면 false 리턴, 숫자가 아니면 true 리턴

#짧은 조건문
 - true || 000 // 우변을 항상 실행하지 않는다
 - false || xxx // 우변을 항상 실행

* 배열
 - 어떤 자료형도 배열 요소가 돌 수 있다
 - ex) var array = [273, 'String', true, function () { }, {}, [273, 103]];

#for in 반복문
 - 일반 for 문, for in 문, forEach함수 이용 가능(이용방법 알아둘 것)
