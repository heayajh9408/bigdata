*CSS 선택자
 oHTML 문서 중 특정 부분을 선택하여, 해당 부분의 속서을 변경하는 목적으로 사용

*선택자들의 종류
 o tag선택자
  - 특정 태그를 선택하여 CSS속성를 적용
 o * 선택자 
  - 전체라는 의미의 * 선택자를 사용하면 문서 전체를 선택
 o id와 class: 
  - 태그의 id와 class를 이용하여 CSS속성을 설정
  - class와 다르게 id는 중복되면 안되고 고유해야함
  - tag, id, class를 혼합하여 CSS에서 속성 설정 가능
 o 속성선택자
  - tag의 속성을 선택하여 CSS의 속성을 설정
  - 선택자[속성]: 지정한 속성을 가지고 있는 요소를 찾아 스타일 적용
  - 선택자[속성=값]: 속성 값이 특정값과 같은 경우를 찾아 스타일 적용
  - 선택자[속성^=값]: 속성 값이 지정한 문자로 시작하는 속성값만 스타일을 적용
  - 선택자[속성$=값]: 속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용
 o 후손 및 자손선택자
 - 후손의 표현방법: div li{}
 - 자손의 표현방법: div>p
 - 후손의 아래의 목록들을 다 포함하고 자손은 하나만 해당 
 o 동위선택자
  - 태그의 동등한 위치를 판단하여 CSS속성을 설정
  - +: 다음에 오는 형제 하나만 
  - ~: 다음에 오는 형제 전부
 o 반응선택자
  - 마우스를 올려 놓았을 때(hover상태)의 반응에 따른 속성을 설정
 o 상태선택자
  - 상태에 따라 CSS속성이 변화하는 설정
  - focus, disable, enable 
 o 구조선택자
  - 구조에 따라 CSS속성이 변화하는 설정
