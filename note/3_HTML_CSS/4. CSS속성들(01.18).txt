*CSS속성들
1. CSS 단위들
 - em, %, px ..
2. url()
 - url(경로) 처럼 사용하며 background-img 속성의 속성값으로 많이 사용
3. background-image
 - 배경을 지정하는 속성으로 
4. display
 - 화면에 어떻게 보이는지를 설정하는 속성
 - block, inline, inline-block(block의 속성을 갖고 옆으로 붙여나감) 을 주로 사용
5. visibility
 - display:none <-> visibility:hidden 
 - display:none은 공간도 사라지면서 안보이는 효과
 - visibility:hidden은 공간을 확보하고 안보이는 효과
6. opacity
 - 투명도를 조절하는 속성
7. margin과 padding
 - 밖여백과 안여백을 설정하는 속성
8. box-sizing
 - 전체 사이즈의 크기를 변경하는 속성
 - ex) box-sizing: border-box = 보더를 박스 안으로
9. 글자 관련 속성들
 - font-famaily(글꼴), font-size(크기), font-style(이텔릭체), font-weight, 
   line-height(행간격 조정 or 세로축 정렬) 등
10. position
 - absoluete: 절대좌표를 지정
 - fixed: 스크롤을 해도 화면샃에서 움직이지 않고 fixed
 - static: 블록형태로 쌓아가며 position값을 주지 않으면 디폴트로 static(static 시 top, left는
   의미가 없다)
 - ralative: 상대적 좌표를 지정
  (###혼합하여 사용 시 z-index가 큰 값이 위로 표현된다###)
11. gradient 속성
 - 2가지 이상의 색을 표현할 때 사용되는 속성
