*입력 양식(form)태그
 o 사용자에게 입력받는 공간을 말하며 입력 양식을 만들 때 사용하는 태그를 말한다
  - action(입력 데이터의 전달 위치 지정), method(입력 데이터의 전달 방식을 선택),
    checkbox(체크박스를 생성), file(파일 입력 양식 생성), hidden(보이지 않게 설정) 
    password(비밀번호 입력 양식), radio(라디오 버튼을 생성), submit(제출버튼생성)
    autofocus(포커스 자동설정), placeholder(입력 형식 힌트제공), pattern(조건을 사용한 일반식 표시)

*공간분할 태그(레이아웃태그)
 o div
  - block 형식으로 공간을 분할하며 하나의 블록형식의 태그므로 개행을 포함한다
 o span
  - inline 형식으로 공간을 분할한다

*시멘틱을 이용한 레이아웃
 o 특정 태그에 의미를 부여해서 검색엔진이 보다 효율적으로 검색하고 유지보수 시 효율적인 방식
 o HTML5 시멘틱 구조 태그
  - header(헤더), nav(내비게이션), aside(사이드에 위치하는 공간), section(여러 중심 내용을 감싸는 공간),
    article(글자가 많이 들어가는 부분), footer(꼬리말)
