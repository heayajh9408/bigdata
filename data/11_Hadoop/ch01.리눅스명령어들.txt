★ Linux 명령어들
1. 현재 디렉토리 위치 : pwd
2. ls -a : 숨긴파일 포함
   ls -F : 파일종류
   ls -R : 하위디렉토리까지
   ls -l : 상세정보
3. cd 폴더 : 폴더이동
   (폴더생략시 사용자홈디렉토리로 이동)
4. 폴더생성 : mkdir 폴더명
   mkdir -p 폴더/폴더 :하위폴더생성
5. 폴더삭제 : rmdir 폴더명
6. 파일보기 : cat, more, 
     less(k이전, j다음, q빠져나감)
     tail(끝부분 보기)
7. 파일/디렉토리 복사 : cp
8. 파일이동 : mv
9. 파일삭제 : rm 파일명
   디렉토리 삭제 : rm -r 디렉토리명
10. 파일링크만들기 : ln
11. 빈파일생성, 수정시간변경 : touch
12. 내용검색 : grep -n
    파일검색 : whereis 파일명
13. 파일찾기 : find
